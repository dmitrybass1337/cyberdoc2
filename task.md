## 📋 Остальные задачи (по приоритету)

### 1. Полный аудит и исправление ошибок проекта

Провести полный аудит сайта **CyberDoc** и рассортировать найденные ошибки:

| Приоритет | Тип ошибок |
|-----------|------------|
| 🔴 Высший | Критические — краш, белый экран, сломанный роутинг |
| 🟠 Высокий | Серьёзные — сломанные функции, ошибки API |
| 🟡 Средний | Умеренные — баги UI, предупреждения в консоли |
| 🟢 Низкий | Минорные — стиль кода, лишние импорты |

Исправить **все** найденные ошибки в порядке убывания приоритета.

### 2. Общие UI/UX исправления

- Убедиться, что **ни один элемент** не перекрывает другой
- Все компоненты рендерятся без визуальных артефактов
- Корректная работа `z-index`, `overflow`, `position` во всём проекте
- Dropdown-меню корректно скроллятся и закрываются

### 3. Инструкция по деплою

Создать пошаговую инструкцию для деплоя сайта на домен:

```
cyberdoc.xyset.ru
```

Инструкция должна включать:

- Команду сборки и переменные окружения
- Настройку Nginx (`server_name cyberdoc.xyset.ru`)
- SSL через Certbot
- Настройку DNS A-записи
- Запуск через PM2 или статический хостинг
